## ğŸŒ Getting Started

1. **Clone the repo:**
   ```bash
   git clone https://github.com/Karthikeya-Thatipamula/jsm_gsap_landing.git
   cd jsm_gsap_landing
    ```
2. **Install dependencies:**
   ```bash
   npm install
   ```
3. **Run the development server:**
    ```bash 
    npm run dev
    ```
4. **Open your browser:**
   Navigate to `http://localhost:5173` to view the app.

5. **Build for production:**
   ```bash
    npm run build
    ```
